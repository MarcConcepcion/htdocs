BarterBayan Set-Up Instructions

requirements:
Xampp
Vscode

Things To do, to work Properly:

1> on phpadmin import database attached
2> on GitHub Repo clone the repository directory "htdocs" folder
3> on htdocs folder file structure should be (\xampp\htdocs\BarterBayan) if GitHub moves it make sure it follow this structure
4> on Vscode open the htdocs folder and run powershell terminal in vscode then call (cd \xampp\htdocs\BarterBayan\frontend)
5> After calling run "npm run dev"

(API integration Specifically Google maps & Cloud Storage cant be achieve yet due to financial reasons)
